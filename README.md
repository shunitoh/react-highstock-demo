### Getting Started

``` shell
$ git clone https://github.com/shunitoh/react-highstock-demo.git
$ npm install
```

### How to Build

```shell
$ gulp build
```

### How to Deploy

```shell
$ node index.js
```

### How to Access

http://localhost:3000/
